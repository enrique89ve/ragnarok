# Ragnarok Card Art Generation Brief

## Style Reference

All art must match the existing Ragnarok card art style exactly:

**Style:** Digital painting, semi-realistic fantasy illustration. NOT anime, NOT pixel art, NOT photorealistic.

**Characteristics:**
- **Lighting:** Dramatic rim lighting, atmospheric glow effects, volumetric fog/mist
- **Color palette:** Rich, saturated — deep golds, crimsons, teals, purples. Dark backgrounds with bright focal points
- **Composition:** Portrait or 3/4 body shot, character centered, facing camera or slightly angled
- **Detail level:** High detail on armor/weapons/faces, painterly backgrounds with bokeh/particle effects
- **Armor style:** Norse knotwork engravings on metal, fur collars/capes, leather straps with gold buckles
- **Magic effects:** Glowing runes, swirling energy (not sparkle/fairy — heavy, elemental energy)
- **Mood:** Serious, mythological weight. Characters look powerful, not cute
- **Background:** Atmospheric (mountains, aurora, ruins, mist, fire) — never plain/solid color
- **Aspect ratio:** Portrait 3:4 (768x1024 or similar)
- **No text, no UI elements, no card frames in the art**

**DO NOT:**
- Use flat/cartoon shading
- Include modern elements
- Make characters smile or look friendly
- Use white/bright backgrounds
- Add watermarks or signatures

## Technical Requirements

| Property | Value |
|----------|-------|
| Resolution | 768x1024 minimum (3:4 portrait) |
| Format | WebP (quality 90+) or PNG |
| File naming | Use format: `{hash}.webp` (8-char hex, e.g., `a1b2-c3d4e5f6.webp`) |
| Destination folder | `client/public/art/` |
| No transparency | Solid painted backgrounds |
| No text/UI | Art only, no card frames or text overlays |
