## How to Use This Document

Each card entry includes:
- **Card ID** — for mapping in `CARD_ID_TO_ART` in `artMapping.ts`
- **Name** — the card's display name
- **Type** — minion, spell, weapon, hero, artifact, armor, secret
- **Rarity** — basic, common, rare, epic, or mythic
- **Mana Cost** — casting cost
- **Stats** — ATK/HP for minions
- **Class** — Neutral or a specific class
- **Race** — creature type (Beast, Dragon, Elemental, etc.)
- **Keywords** — card abilities (Taunt, Divine Shield, etc.)
- **Description** — card text from the registry
- **AI Art Prompt** — suggested prompt for generating art

Art specs: **512×512 WebP**, transparent or dark background, centered subject.
File naming: `{4-hex}-{8-hex}.webp` (hash format) or descriptive name.
After generating, add mapping to `CARD_ID_TO_ART` in `client/src/game/utils/art/artMapping.ts`.
